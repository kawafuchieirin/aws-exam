# AWS Global Accelerator の要点

> 重要度: ○ ／ ANS-C01 第1分野で頻出。L4 のグローバル可用性・固定 IP・高速フェイルオーバーが論点。CloudFront との使い分けが定番。

## これは何
- **2 つの静的 Anycast IP** を単一エントリポイントとして提供する **L4（TCP/UDP）** ネットワークサービス。
- 最寄りエッジから **AWS グローバルバックボーン**経由で最適リージョンの ALB/NLB/EC2/EIP へ届ける。
- キャッシュしないプロキシ。HTTP に限定されない（ゲーム・IoT・VoIP・MQTT 等）。

## 試験頻出ポイント
- **静的 Anycast IP は IPv4 で 2 個**（デュアルスタックは計 4 個）。アクセラレーター存在中は保持、**disable でも保持・delete で失効**。**BYOIP（IPv4）**可。
- **2 つのネットワークゾーン**から 1 IP ずつ提供 → 片方がブロックされても他方へリトライ可。
- **AWS バックボーン経由**で輻輳の少ない経路を通る（パフォーマンス・可用性向上）。
- **クライアント IP 保持**: ALB/EC2 エンドポイントへは実 IP を保持でき、SG でクライアント IP 制御可。
- **トラフィックダイヤル**＝エンドポイントグループ（リージョン）単位の流量割合（0–100%）。**エンドポイント重み**＝エンドポイント単位の比率（0–255）。粒度の違いが頻出。
- **クライアントアフィニティ**: 既定 **None＝5 タプル**で分散／**Source IP＝2 タプル**で同一クライアントを同一エンドポイントへ固定（セッション維持）。
- **高速フェイルオーバー**: ヘルスチェック間隔 **10/30 秒**（既定 30）、異常しきい値既定 3 回。不健全リージョンを **数十秒で自動切替**。固定 IP のまま切り替わるため **DNS TTL に依存せず Route 53 フェイルオーバーより速い**。
- **Standard**＝ヘルス・位置・重みで最適ルーティング（NLB/ALB/EC2/EIP）。**Custom routing**＝決定論的に特定 EC2（VPC サブネットのプライベート IP）へ割当（ゲームセッション固定用、デュアルスタック非対応）。
- **Accelerated Site-to-Site VPN** は **TGW にアタッチした VPN でのみ**有効化可（VGW 直アタッチ不可）。
- **API Gateway** は直接エンドポイント不可（NLB＋VPC リンク経由で固定 IP 化）。**Shield Advanced** の保護対象。

## CloudFront との使い分け
| 観点 | Global Accelerator | CloudFront |
|---|---|---|
| レイヤ | **L4（TCP/UDP）** | **L7（HTTP/HTTPS）** |
| キャッシュ | なし（プロキシ） | あり（エッジキャッシュ） |
| エントリ | **2 つの静的 Anycast IP** | ディストリビューションのドメイン名 |
| プロトコル | 非 HTTP 含む（ゲーム/IoT/VoIP/MQTT） | HTTP/HTTPS 中心 |
| TLS | 終端しない（パススルー） | エッジで終端 |
| フェイルオーバー | リージョン障害に高速・固定 IP | オリジングループ（HTTP） |
| WAF | 非対応 | 対応 |

> **キャッシュ・HTTP・WAF が必要** → [CloudFront](cloudfront.md)。**非 HTTP・固定 IP・L4 高速フェイルオーバー** → Global Accelerator。

## よく問われる上限・注意点（ひっかけ）
- 静的 IP は「リージョンごとに 1 個」ではなく **全リージョンで同一の Anycast IP 2 個**（Anycast なので同一 IP が広告される）。
- **disable では IP は失効しない**（保持される）。失効するのは **delete** のみ。
- ALB/NLB は **そのリソース自身のヘルスチェック結果を GA が尊重**。EC2/EIP は **GA のヘルスチェック**を使う。
- **トラフィックダイヤルとエンドポイント重みを混同しない**（ダイヤル＝グループ単位、重み＝エンドポイント単位）。
- Custom routing のエンドポイントは **VPC サブネット内 EC2 のプライベート IP のみ**（NLB/ALB は不可）。
- コストは **固定時間料金＋データ転送プレミアム（DT-Premium）**。CloudFront のようなキャッシュ割引はない。
